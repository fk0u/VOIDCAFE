# 🌑 VOIDCAFE

> **The darkest K-pop fan forum. Enter the void.**

## ✨ Features

### 🎯 Core
- **Full Forum Experience** — Create threads, reply with nested comments (3 levels), like, bookmark, share
- **10 K-pop Communities** — AESPA, BTS, BLACKPINK, TWICE, Stray Kids, NewJeans, (G)I-DLE, IVE, LE SSERAFIM, ATEEZ
- **18+ Seed Threads** — Realistic K-pop discussions pre-loaded
- **User Profiles** — Avatar upload, bio editing, bias list, activity tabs

### ⚡ Technical
- **TanStack Router** — File-based routing with automatic code splitting
- **TanStack Query v5** — Data fetching with infinite scroll, optimistic updates, mutations, cache invalidation
- **Zustand** — Persistent global state (auth, preferences)
- **localStorage** — Full data persistence (no backend needed)
- **TypeScript** — Strict mode, zero errors

### 🎨 Visual
- **Aceternity-style UI** — Spotlight effect, 3D cards, magnetic buttons, animated tooltips, background beams
- **Glitch Animation** — CSS-based glitch text effect on VOIDCAFE logo
- **Glassmorphism** — Backdrop blur navbar, modals, and cards
- **Horror Particles** — Subtle floating particles across the page
- **Metallic Gradients** — Chrome shimmer text effects
- **Scanline Overlays** — Retro CRT scanline effect on cards
- **Neon Glow** — Purple, cyan, and red neon accents throughout

### 🔧 Polish
- **Keyboard Shortcuts** — `N` = New Post, `/` = Search, `Escape` = Close
- **Toast Notifications** — Animated success/error/info/warning toasts
- **Loading Skeletons** — Beautiful shimmer loading states
- **Infinite Scroll** — Seamless thread loading with intersection observer
- **Responsive Design** — Mobile bottom nav, collapsible sidebar
- **PWA Ready** — Manifest included
