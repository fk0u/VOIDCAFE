# 🚀 Getting Started

Welcome to **VOIDCAFE** — the dark cyber-metallic K-pop fan forum inspired by AESPA's "metal tech" aesthetic and subtle eerie horror vibes.

---

## 🛠️ Prerequisites

Make sure you have the following installed:
- **Node.js** (v18 or higher recommended)
- **npm** (v9 or higher)

---

## 📦 Installation & Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```
   *The application will boot at **http://localhost:5173**.*

3. **Build for production**:
   ```bash
   npm run build
   ```
   *This compiles TypeScript and builds minimized chunks into the `/dist` directory.*

4. **Preview the production build locally**:
   ```bash
   npm run preview
   ```

## ⌨️ Global Keyboard Shortcuts

VOIDCAFE supports global hotkeys to navigate the portal quickly.
*(Pressing these keys will not trigger if your cursor is focused inside an input field or textarea).*

| Key | Action |
|-----|--------|
| `N` / `n` | Opens the **Create Thread** modal |
| `/` | Opens the **Global Search Overlay** |
| `` ` `` (Backtick / `~`) | Toggles the **VOID-Terminal Console** |
| `Escape` | Closes any active modal, blur inputs, and disconnects terminal |
